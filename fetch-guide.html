Fetch Guide Data

Enter a Rexby guide ID to fetch all its thingsToDo data. The result will download as a JSON file.

How to find a guide ID:
Open any guide on Rexby, open DevTools Network tab, look for GraphQL requests — the guide ID is in the variables.

After downloading:
1. Save the JSON file to data/{guideId}.json
2. Add an entry to data/guides.json
3. Commit and push — the site updates automatically
